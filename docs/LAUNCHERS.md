# CGCUT 项目启动器说明

本项目提供了多个启动器脚本，方便您快速启动开发环境。

## 启动器脚本说明

### 1. `run-dev.bat` (推荐)
- 直接运行开发服务器
- 保持命令行窗口打开，显示实时日志
- 访问地址：http://localhost:5173
- 按 `Ctrl+C` 停止服务器

### 2. `start-dev.bat`
- 启动开发服务器（在新窗口中运行）
- 自动打开浏览器访问应用
- 按提示关闭服务器

### 3. `run-electron.bat` (推荐)
- 直接运行Electron桌面应用
- 保持命令行窗口打开，显示实时日志
- 按 `Ctrl+C` 停止应用

### 4. `start-electron.bat`
- 启动Electron桌面应用（在新窗口中运行）
- 按提示关闭应用

### 5. `start-dev.ps1`
- PowerShell版本的启动脚本
- 自动打开浏览器访问应用

### 6. `build-app.bat`
- 构建生产版本的应用

## 使用建议

1. 对于开发调试：使用 `run-dev.bat` 或 `run-electron.bat`，可以看到实时日志
2. 对于快速启动：使用 `start-dev.bat`，会自动打开浏览器
3. 如果需要桌面应用体验：使用 `run-electron.bat`

## 注意事项

- 首次运行会自动安装依赖（如果尚未安装）
- 确保您的系统已安装Node.js和npm
- 如果遇到权限问题，可能需要以管理员身份运行脚本
