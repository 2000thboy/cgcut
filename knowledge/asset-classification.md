# 动漫/视频素材分类体系

> 本知识库用于 CLIP 打标和素材检索的标签体系定义
> 更新时间：2026-01-15

---

## 1. 核心分类维度

### 1.1 景别分类 (Shot Type)

| 标签 | 英文             | 说明             | CLIP 标签  |
| ---- | ---------------- | ---------------- | ---------- |
| 特写 | Extreme Close-up | 眼睛、嘴唇等细节 | `特写镜头` |
| 近景 | Close-up         | 肩部以上         | `近景镜头` |
| 中景 | Medium Shot      | 腰部以上         | `中景镜头` |
| 全景 | Wide Shot        | 全身及环境       | `全景镜头` |
| 远景 | Long Shot        | 大环境           | `远景镜头` |

### 1.2 场景类型 (Scene)

| 标签 | 子类别                   | CLIP 标签     |
| ---- | ------------------------ | ------------- |
| 室内 | 办公室、卧室、教室、餐厅 | `室内场景`    |
| 室外 | 街道、公园、广场         | `室外场景`    |
| 自然 | 森林、海滩、山脉、草原   | `自然风景`    |
| 城市 | 都市、高楼、霓虹         | `城市`        |
| 特殊 | 战场、太空、异世界       | `fantasy场景` |

### 1.3 主体类型 (Subject)

| 标签     | 说明           | CLIP 标签  |
| -------- | -------------- | ---------- |
| 人物单体 | 单人特写或动作 | `人物`     |
| 人物群体 | 多人互动场景   | `群体场景` |
| 面部表情 | 情绪表达特写   | `面部特写` |
| 手部动作 | 手势、操作特写 | `手部特写` |
| 空镜     | 无人物的环境   | `无人场景` |
| 物体     | 道具、物品特写 | `物体特写` |

### 1.4 动作类型 (Action)

| 标签 | 包含动作         | CLIP 标签  |
| ---- | ---------------- | ---------- |
| 静态 | 站立、坐着、静止 | `静坐`     |
| 行走 | 走路、漫步       | `行走`     |
| 奔跑 | 跑步、追逐       | `奔跑`     |
| 战斗 | 打斗、格斗、枪战 | `战斗`     |
| 对话 | 交谈、对视       | `对话交流` |
| 工作 | 操作、处理       | `工作`     |

### 1.5 情绪分类 (Emotion)

| 情绪 | 视觉特征         | CLIP 标签 | 对应分镜情绪 |
| ---- | ---------------- | --------- | ------------ |
| 紧张 | 快速镜头、不稳定 | `紧张`    | 紧张         |
| 平静 | 稳定、柔和光线   | `平静`    | 平静         |
| 激动 | 强烈动作、高对比 | `激动`    | 喜悦/愤怒    |
| 悲伤 | 低饱和、慢动作   | `悲伤`    | 悲伤         |
| 快乐 | 明亮、活泼       | `快乐`    | 喜悦         |
| 恐惧 | 阴影、暗调       | `恐惧`    | 恐惧         |
| 愤怒 | 强对比、快节奏   | `愤怒`    | 愤怒         |

### 1.6 光线/色调 (Lighting)

| 标签   | 特征         | CLIP 标签 |
| ------ | ------------ | --------- |
| 明亮   | 高亮度、清晰 | `明亮`    |
| 昏暗   | 低亮度、阴影 | `昏暗`    |
| 自然光 | 日光、环境光 | `自然光`  |
| 人工光 | 灯光、霓虹   | `人工光`  |

### 1.7 时间段 (Time)

| 标签 | 视觉特征       | CLIP 标签 |
| ---- | -------------- | --------- |
| 白天 | 明亮、自然色彩 | `白天`    |
| 夜晚 | 黑暗、灯光     | `夜晚`    |
| 黄昏 | 暖色调、剪影   | `黄昏`    |
| 清晨 | 柔和、薄雾     | `清晨`    |

| 热血 | 强烈动作、特效 | `热血`, `战斗`, `激动`   |

| 科幻 | 未来感、科技   | `科幻`, `城市`, `人工光` |

### 2.2 常见动漫场景

| 场景类型 | 典型环境         | 推荐标签组合                   |
| -------- | ---------------- | ------------------------------ |
| 校园     | 教室、走廊、操场 | `室内场景`, `群体场景`, `日常` |
| 战斗     | 战场、擂台、街头 | `战斗`, `激动`, `动作镜头`     |

## 3. 标签匹配规则

### 3.1 分镜 → 素材 匹配优先级

1. **情绪匹配** (权重: 40%)

   - 分镜情绪 → 素材情绪标签
   - 紧张 → 紧张/恐惧, 平静 → 平静/日常

2. **景别匹配** (权重: 30%)

   - [特写] → 特写镜头/近景镜头
   - [全景] → 全景镜头/远景镜头

3. **内容匹配** (权重: 20%)

   - 关键词提取 → CLIP 向量搜索
   - "办公室" → 室内场景+办公室

4. **时长匹配** (权重: 10%)
   - 短镜头(1-3 秒) → 快速动作素材
   - 长镜头(6 秒+) → 稳定场景素材

### 3.2 CLIP 搜索策略

```python
# 复合查询示例
query_strategy = {
    "primary": "分镜描述文本",     # 语义搜索
    "filters": {
        "emotion": ["紧张", "恐惧"],  # 情绪过滤
        "shot_type": ["特写", "近景"], # 景别过滤
    },
    "threshold": 0.25,  # 相似度阈值
    "top_k": 5          # 返回数量
}
```

## 4. 标签体系维护

### 4.1 标签添加原则

- 每个素材至少 3 个标签
- 必须包含：景别 + 情绪 + (场景或主体)
- 可选：动作 + 光线 + 时间

### 4.2 标签命名规范

- 使用中文标签（便于分镜匹配）
- 标签简洁（2-4 个字）
- 避免歧义（战斗 vs 打架 → 统一用"战斗"）

### 4.3 自动标签流程

```
视频文件
  ↓
CLIP提取关键帧
  ↓
CLIP向量编码
  ↓
与预定义标签匹配
  ↓
置信度>0.2的标签保留
  ↓
VLM生成描述（补充）
  ↓
最终标签集合
```

## 5. 素材质量评估

### 5.1 技术指标

| 指标   | 推荐值   | 说明         |
| ------ | -------- | ------------ |
| 分辨率 | ≥720p    | 保证清晰度   |
| 帧率   | 24-30fps | 标准电影帧率 |
| 时长   | 2-15 秒  | 适合分镜使用 |
| 格式   | MP4/MOV  | 兼容性好     |

### 5.2 内容质量

- 画面稳定（无过度抖动）
- 主体清晰（焦点明确）
- 内容完整（动作有始有终）
