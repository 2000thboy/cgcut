## 1. 分镜头脚本定义

**分镜头脚本（Shot List / Storyboard）**是将文字剧本转化为可拍摄的视觉镜头序列的蓝图。它是导演、摄影师、剪辑师、演员等所有制作人员的统一工作依据。

### 核心作用
- 📐 **施工蓝图**：明确每个镜头的视觉呈现
- 🎬 **拍摄计划**：规划拍摄顺序和资源调度
- 🤝 **沟通工具**：统一全体制作人员的理解
- ⏱️ **时长控制**：精确计算影片节奏

## 2. 镜头景别分类（Shot Size）

### 2.1 远景镜头（Long Shot / LS）
- **视觉范围**：大范围场景，人物较小
- **应用场景**：
  - 建立场景位置和环境氛围
  - 展示人物与环境的空间关系
  - 大场面调度和群体活动
- **情绪表达**：开阔、孤独、渺小、宏大
- **典型时长**：5-12秒

**示例**：
```
[远景] 城市夜景，高楼林立，霓虹闪烁，街道车流如织
```

### 2.2 全景镜头（Wide Shot / WS）
- **视觉范围**：人物全身及周围环境
- **应用场景**：
  - 展示人物完整动作
  - 多人对话场景
  - 动作戏的完整呈现
- **情绪表达**：全局观感、空间关系、动态节奏
- **典型时长**：4-10秒

**示例**：
```
[全景] 办公室内，三人围坐会议桌，主角站起身指着白板
```

### 2.3 中景镜头（Medium Shot / MS）
- **视觉范围**：人物膝盖/腰部以上
- **应用场景**：
  - 日常对话场景
  - 展示肢体语言和互动
  - 平衡环境与人物
- **情绪表达**：自然交流、情感互动、日常生活
- **典型时长**：3-8秒

**示例**：
```
[中景] 主角坐在餐桌前，拿起咖啡杯，眼神望向窗外
```

### 2.4 近景镜头（Close-up / CU）
- **视觉范围**：人物肩部以上或物体局部
- **应用场景**：
  - 展示面部表情
  - 强调重要道具
  - 情绪高潮时刻
- **情绪表达**：亲密、紧张、专注、情感爆发
- **典型时长**：2-6秒

**示例**：
```
[近景] 主角面部，眉头紧锁，汗水从额头滑落
```

### 2.5 特写镜头（Extreme Close-up / ECU）
- **视觉范围**：眼睛、嘴唇、手指等极小细节
- **应用场景**：
  - 极致的情感表达
  - 关键细节的强调
  - 悬念制造
- **情绪表达**：极度紧张、恐惧、愤怒、惊讶
- **典型时长**：1-4秒

**示例**：
```
[特写] 主角眼睛，瞳孔急剧收缩，倒映出门外的黑影
```

## 3. 摄像机角度（Camera Angle）

### 3.1 平视角度（Eye Level）
- **效果**：中性、客观、日常视角
- **应用**：大部分对话和叙事场景

### 3.2 俯拍角度（High Angle）
- **效果**：弱小、压抑、被动、脆弱
- **应用**：展示角色的弱势地位

### 3.3 仰拍角度（Low Angle）
- **效果**：强大、威严、压迫、权威
- **应用**：展示角色的权力或威胁感

### 3.4 鸟瞰角度（Bird's Eye View）
- **效果**：全局俯瞰、命运感、孤立无援
- **应用**：大场景调度或情感高潮

### 3.5 荷兰角度（Dutch Angle / Canted Angle）
- **效果**：不安、混乱、失衡、紧张
- **应用**：悬疑、惊悚场景

## 4. 镜头运动（Camera Movement）

### 4.1 推镜头（Push In / Dolly In）
- **效果**：强调、聚焦、情绪递增
- **应用**：引导观众注意力到关键点

### 4.2 拉镜头（Pull Out / Dolly Out）
- **效果**：揭示、扩展、疏离感
- **应用**：展现更大的环境信息

### 4.3 摇镜头（Pan）
- **效果**：跟随、扫视、空间转换
- **应用**：展示横向空间关系

### 4.4 移动镜头（Tracking / Dolly）
- **效果**：流畅跟随、动态叙事
- **应用**：跟随人物移动或营造流动感

### 4.5 升降镜头（Crane / Boom）
- **效果**：视角变化、宏大感
- **应用**：开场/结尾的气势营造

## 5. 情绪标注系统

### 5.1 核心情绪分类

| 情绪 | 镜头特征 | 典型景别 | 节奏 |
|------|----------|----------|------|
| **紧张** | 快速切换、特写、荷兰角度 | 特写、近景 | 快节奏 |
| **焦虑** | 手持摄影、晃动、不稳定构图 | 近景、中景 | 不规则 |
| **恐惧** | 阴影、低照度、仰拍/俯拍 | 特写、全景 | 突变 |
| **释然** | 稳定镜头、柔和光线、缓慢移动 | 全景、中景 | 舒缓 |
| **平静** | 静止镜头、平衡构图、自然光 | 全景、远景 | 慢节奏 |
| **愤怒** | 强烈对比、快速推进、冲击感 | 近景、特写 | 爆发性 |
| **悲伤** | 长镜头、缓慢移动、冷色调 | 近景、中景 | 延长 |
| **喜悦** | 明亮、跳跃式剪辑、动态 | 全景、中景 | 活泼 |

## 6. 时长估算标准

### 6.1 基于景别的时长
- **特写/近景**：2-5秒（快速传递情绪）
- **中景**：3-8秒（平衡叙事）
- **全景**：5-10秒（建立空间）
- **远景**：6-12秒（营造氛围）

### 6.2 基于内容的调整
- **纯视觉镜头**：+20%时长（让观众读取信息）
- **包含对白**：根据台词长度计算（每字约0.3秒）
- **动作场景**：-10%时长（快节奏）
- **情感高潮**：+30%时长（给予情绪发酵空间）

### 6.3 时长计算公式
```
基础时长 = 景别时长 + 对白时长
实际时长 = 基础时长 × 情绪系数 × 节奏系数
```

## 7. 分镜拆解原则

### 7.1 完整性原则
每个镜头应该是：
- ✅ 一个完整的视觉单元
- ✅ 从开始到结束的连续动作
- ✅ 具有明确的开始和结束点

### 7.2 主体明确原则
每个镜头必须有：
- 👤 明确的视觉主体（人、物、景）
- 🎯 清晰的焦点
- 📍 可识别的空间位置

### 7.3 情绪传递原则
通过镜头语言传递：
- 😊 角色的情感状态
- 🎭 场景的情绪氛围
- 📖 叙事的情感走向

### 7.4 节奏控制原则
- ⚡ 紧张场景：短镜头、快切换
- 🌊 平静场景：长镜头、慢节奏
- 💥 高潮场景：节奏变化、反差强烈

## 8. 分镜编写模板

### 8.1 标准格式
```

```

### 8.2 优秀示例


## 9. 场景拆解策略

### 9.1 单场景拆解标准
- **对话场景**：3-6个镜头
  - 建立镜头（全景）
  - 人物特写（正反打）
  - 反应镜头（情绪捕捉）
  
- **动作场景**：5-10个镜头
  - 建立动作全貌（全景）
  - 关键动作细节（近景、特写）
  - 结果展示（中景）
  
- **情感场景**：4-8个镜头
  - 环境渲染（远景、全景）
  - 情绪递进（中景→近景→特写）
  - 情感高潮（特写）

### 9.2 多场景串联原则
- 📍 **空间转换**：用定场镜头明确新场景
- 🎬 **时间跳跃**：用淡入淡出或转场镜头
- 🔗 **叙事连贯**：保持情绪和节奏的逻辑延续

## 10. 常见错误与避免

### ❌ 错误1：镜头过长
- **问题**：单个镜头超过15秒，观众注意力分散
- **解决**：拆分为2-3个子镜头，保持视觉变化

### ❌ 错误2：景别单调
- **问题**：连续多个相同景别，缺乏视觉节奏
- **解决**：交替使用不同景别，营造层次感

### ❌ 错误3：情绪不明
- **问题**：镜头描述缺少情感指向
- **解决**：明确标注每个镜头的情绪标签

### ❌ 错误4：拆分过细
- **问题**：将一个动作拆成过多碎片
- **解决**：保持镜头的完整性，一个动作一个镜头

## 11. 实战拆解流程

### Step 1: 场景识别
- 读取剧本中的场景标识（INT./EXT.）
- 确定场景地点和时间
- 分析场景的主要叙事功能

### Step 2: 确定镜头数量
- 对话场景：3-6镜
- 动作场景：5-10镜
- 情感场景：4-8镜
- 过渡场景：1-3镜

### Step 3: 设计镜头序列
1. 建立镜头（定场）
2. 主体镜头（叙事核心）
3. 反应镜头（情感呼应）
4. 细节镜头（强调重点）
5. 结束镜头（承接下场）

### Step 4: 标注镜头信息
- 镜头类型（景别）
- 视觉主体
- 动作描述
- 情绪标签
- 时长估算

### Step 5: 整体审查
- 检查镜头逻辑连贯性
- 确认情绪弧线流畅
- 验证时长合理性
